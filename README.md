# Луzа и Шары - анализ запуска бильярдного клуба в Москве

Курсовая по дисциплине «Наука о данных, spring26» (продвинутый поток).

## Идея

Нишевая бильярдная в Москве с упором на качественный сервис, атмосферу и
удобное бронирование. Цель проекта - понять, где открывать, под какую
аудиторию, с какой командой и каким бюджетом.

## Источники данных

| Источник | Метод | Что даёт |
|---|---|---|
| 2GIS Catalog API | requests + JSON | Конкуренты: рейтинги, отзывы, координаты |
| sob.ru | requests + BeautifulSoup | Аренда коммерческой недвижимости |
| OpenStreetMap (Overpass API) | requests + JSON | Координаты бильярдных, дополнение к 2GIS |
| Nominatim | REST API | Уточнение округа по координатам |
| Zoon.ru | Selenium | Досуговые заведения, негативные отзывы |
| Yandex.Карты | Selenium (CSV выгрузка) | Отзывы по бильярдным клубам |
| Trudvsem | requests + JSON API | Вакансии под бильярдный клуб |
| Billiard1 / Ozone Billiards | requests + BeautifulSoup | Цены на оборудование |

Итого 8 различных источников при минимуме по требованиям 2-3.

## Требования продвинутого потока

| Требование | Где реализовано |

## Запуск

```bash
pip install -r requirements.txt
jupyter notebook analysis.ipynb
```

Все собранные данные уже лежат в `data/`. Первый запуск использует кэш,
парсеры заново запустятся только если удалить соответствующий CSV.

Дашборд открывается на `http://localhost:8050` (последняя ячейка ноутбука).

## Гипотезы

Все тесты в `src/hypotheses.py`, вызываются из ноутбука одной командой
`hypotheses.run_all_tests(...)`:

1. **t-test**: рейтинги бильярдных vs других досуговых заведений
2. **chi-squared**: связь типа заведения и категории жалоб
3. **ANOVA**: разница арендных цен между округами Москвы
4. **Spearman**: корреляция рейтинга и количества отзывов
5. **Bootstrap CI**: доверительные интервалы стоимости стартового комплекта

## Состав дашборда

- Карта - 2GIS и OSM с фильтрами по округу и источнику
- Конкуренты - распределение рейтингов, топ-10, по округам
- Аренда и досуг - медианная аренда + плотность досуга
- Отзывы конкурентов - категории жалоб, тиеры рейтинга, частотный анализ слов
- Где открывать - opportunity score, scatter спрос/предложение
- План действий - автогенерированные рекомендации из данных
- Аудитория - темы отзывов, тональность, портреты ЦА
- Рынок труда - зарплаты, ФОТ, графики работы, топ работодателей
- Оборудование - категории, сегменты, стартовый комплект
