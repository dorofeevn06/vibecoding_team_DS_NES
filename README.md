# Луzа и Шары - эксплуатационная часть проекта

Курсовая по дисциплине «Наука о данных, spring26» (продвинутый поток).
Эта часть проекта отвечает за три направления: целевая аудитория, рынок
труда и цены на оборудование для запуска бильярдного клуба.

## Автор

Михайлов Артур (Avmikhajlov)

## Что в проекте

Pipeline из трёх стадий:

1. **Парсинг** (`src/parsers/`) - сбор сырых данных
2. **Обработка** (`src/processing/`) - чистка, разметка, агрегация
3. **Анализ** (`src/analysis/`) - сводки, графики, отчёты

Плюс единый дашборд из 3 вкладок и тонкий отчётный ноутбук.

## Источники данных

| Источник | Метод | Что даёт |
|---|---|---|
| Яндекс.Карты | Selenium | Отзывы по 135 бильярдным и досуговым заведениям Москвы (≈7500 отзывов) |

## Запуск

```bash
pip install -r requirements.txt
jupyter notebook analysis.ipynb
```

Очищенные CSV уже лежат в `data/processed/`. Запуск парсеров заново нужен
только если хотите обновить данные.

Каждый аналитический модуль можно запускать и отдельно:

```bash
python src/analysis/audience_analysis.py
python src/analysis/jobs_analysis.py
python src/analysis/equipment_analysis.py
```

Результаты лягут в `results/`: HTML-графики, CSV-таблицы и текстовые
отчёты со сводками.

## Дашборд

Три вкладки, открывается через ноутбук на `http://localhost:8051`:

- **Аудитория** - что обсуждают в отзывах, тональность по темам, главные
  боли клиентов, заведения с наибольшим числом отзывов.
- **Рынок труда** - медианные зарплаты по позициям, зарплатные коридоры,
  прогноз ФОТ для типового клуба, графики работы, топ работодателей.
- **Оборудование** - медианная цена по категориям, распределение по
  ценовым сегментам, стоимость стартового комплекта на 8 столов, разброс
  цен внутри категорий.

## Что показал анализ

- **Аудитория**: бильярдный клуб воспринимается как формат вечернего
  досуга, а не чисто спортивное место. Чаще всего обсуждают сервис,
  атмосферу и столы. Главные боли - бронирование и общение с персоналом.
- **Кадры**: на штат из 16 человек уходит около 1,17 млн ₽/мес брутто.
  Самые дорогие позиции - SMM и технический специалист, экономить на них
  рискованно.
- **Оборудование**: стартовый комплект на 8 столов в среднем сегменте
  обходится в ~1,9 млн ₽. Основная статья - столы и сукно (60-70%
  бюджета). Премиум-столы оправданы только под турниры (1-2 на зал).
